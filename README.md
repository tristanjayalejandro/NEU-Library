# NEU Library Visitor Log

A library visitor log system for New Era University built with HTML, CSS, JavaScript, PHP, and MySQL.

## Live App

[https://neulibrary.infinityfreeapp.com](https://neulibrary.infinityfreeapp.com/index.html)

## Login

Use your @neu.edu.ph Google account to sign in.

## Features

- Google OAuth login (authorized accounts only)
- Student self-registration and check-in / check-out kiosk
- Admin dashboard with live visitor statistics
- Filter stats by date range, college, visit reason, and visitor type
- Student management with block / unblock access control
- Role-based access: switch between Admin and User view

## Tech Stack

HTML, CSS, JavaScript, PHP, MySQL, Google OAuth 2.0
